# Hardwood 26

A self-contained, fan-made browser basketball game with all 30 pro teams, modern and all-time lineup modes, CPU opponents, timed shooting, passing, stealing, fouls, free throws, sprint stamina, rebounds, shot clock, quarters, touch controls, and a full matchup menu.

Open `index.html` in a browser to play.

## Android

When hosted over HTTPS, open the game in Chrome on Android and choose **Install on Android**. The game runs fullscreen in landscape and caches its files for offline play.

## Controls

- `WASD`: move
- `Shift`: sprint
- `Space`: hold and release to shoot
- `R`: dunk when close to the basket
- `Q`: pass
- `F`: steal
- `E`: switch defender
- `Esc`: pause

This project is an unofficial fan creation and is not affiliated with or endorsed by the NBA or its teams.
